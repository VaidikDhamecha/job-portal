# job-portal
A Job Portal Web Application built using PHP, MySQL, Bootstrap 5, and JavaScript. It allows Job Seekers to search &amp; apply for jobs, manage their profile, and upload resumes, while Employers can post job listings, manage applicants, and showcase company details.
